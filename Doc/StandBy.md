## [System Keys Quick Guide](./HelpIndex.md)
  - `<SPACE>`     - Left-Click  
  - `<ENTER>`     - Right-Click  
  - `<F1>`        - Help
  - `<CTRL-m>`    - Personal Menu
  - `<ESC>`       - Save  & Exit
  - `<SHIFT-ESC>` - Abort & Exit
  - `<WINDOWS>`   - Pull-Down Menu
(Works Everywhere)
